## Hash_Identifier

Tool creato seguendo la roadmap dei progetti riguardo la CyberSecurity: https://github.com/CarterPerez-dev/Cybersecurity-Projects

## A cosa serve?

Hash_Identifier è un tool da riga di comando usato per identificare il tipo di hash passato come argomento.

## Linguaggio di Programmazione 

Hash_Identifier è completamente scritto in C++.

## Hash identificabili 

### MD5

### SHA-1

### SHA-256

### SHA-512

### bcrypt

### Argon2id

### Apache MD5

### MySQL5

## Network-Analyzer 

Tool di analisi di rete scritto in c++ utilizzando la libreria libpcap.

link alla documentazione della libreria: https://www.tcpdump.org/pcap.html


link alla roadmap dei progetti riguardo la CyberSecurity: https://github.com/CarterPerez-dev/Cybersecurity-Projects

## Requisiti di Sistema e Installazione

Questo progetto richiede le librerie di sviluppo  **Npcap**  per catturare e analizzare il traffico di rete.

Scarica e installa Npcap Installer dal sito ufficiale: https://npcap.com/#download

Scarica il file .zip di Npcap SDK dalla stessa pagina di download.
